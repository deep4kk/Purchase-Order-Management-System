# Purchase Order Management System

A comprehensive Google Apps Script system for managing purchase orders with multi-level approval and inventory integration.

## Features

- **Multi-Level Approval**: Manager + Finance approval for POs
- **Inventory Integration**: Auto-update inventory on PO receipt
- **Vendor Management**: Track vendor details and performance
- **PO Status Tracking**: Draft → Submitted → Manager → Finance → Approved → Ordered → Received
- **PDF Generation**: Generate printable PO documents

## Setup

1. Create Google Sheet with: Users, Vendors, PurchaseOrders, POLineItems, AuditLog
2. Deploy as Web App
